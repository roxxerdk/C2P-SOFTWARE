# Hole Tolerancing and Sizing Standards

This standard specifies manufacturing rules for drilling, tapping, and reaming holes.

## 1. General Sizing Guidelines
- **Aspect Ratio Limit**: Hole depth should not exceed 5x the diameter (5D rule) to prevent drill deflection and breakage.
- **Pilot Hole Requirement**: Any hole diameter greater than 0.500" requires a pilot hole (typically 0.125" or 0.250").

## 2. Standard Tolerances
- **As-Drilled Holes**: +0.005" / -0.001"
- **Reamed Holes**: +0.0005" / -0.0000"
- **Tapped Holes**: Class 2B fit general, Class 3B fit for aerospace applications.
